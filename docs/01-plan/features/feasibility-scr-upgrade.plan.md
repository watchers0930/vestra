# VESTRA 사업성 분석 SCR 보고서 동일 출력물 고도화 기획서

> **버전**: v1.1.0
> **작성일**: 2026-03-22 (v1.1.0 업데이트)
> **참조**: SCR2025-사-12 울산 우정동 주상복합 신축사업 (79p, 표64개, 그림23개)

---

## 1. 목표

사용자가 사업계획서, 재무분석표, 투자계획서 등을 업로드하면 **SCR 서울신용평가의 사업성평가보고서와 동일한 목차/상세구조/표/그래프**를 갖춘 보고서를 자동 생성한다.

### 현재 vs 목표

| 구분 | 현재 (6장 구조) | 목표 (SCR 동일 5장+부록) |
|------|-------------|-------------------|
| I. 사업 개요 | 요약만 | 사업개요/구조도/일정/분양계획/자금조달/토지현황 |
| II. 사업주체 분석 | **없음** | 회사개요/사업분석/수익성/재무안정성/현금흐름 |
| III. 시장 환경 분석 | 벤치마크 1줄 | 규제/인구·사업체/부동산시장(주택보급률/거래량/건축연령/공급/미분양) |
| IV. 분양가 적정성 검토 | 괴리율만 표시 | 입지여건/인근개발/시설개요/매매사례13건/분양사례10건/적정성의견 |
| V. 원리금상환가능성 | V-Score만 | 재무분석(전제/사업수지/자금수지) + 월별자금수지 + 시나리오 + BEP |
| Appendices | **없음** | 정책/규제지역/HUG/금리/지수/인근개발 |
| 표 개수 | ~10개 | **64개** |
| 그림/차트 | ~3개 | **23개** |
| 총 페이지 | ~8p | **60~80p** |

## 2. SCR 보고서 전체 목차 (타겟 구조)

### 전문부 (Front Matter)
- 표지 (사업명, 시행사, VESTRA 로고)
- 보고서 용도 / 분석의 전제 / 유의사항
- 분석 담당자 정보
- **전체 목차** / **표 목차** / **그림 목차** / **부록 목차**

### I. 사업 개요
| 소항목 | 주요 표/그림 | 데이터 소스 |
|--------|-----------|----------|
| 1. 사업 개요 | 표: 본 계획사업 개요 (사업명/위치/지구/시공사/면적/용적률/규모/용도/공사기간) + 조감도 | 사용자 입력 |
| 2. 사업 구조 | 그림: 사업 구조도 (수분양자↔시행·시공사↔대주단↔신탁사) | 사용자 입력 |
| 3. 사업 일정 | 표: 사업일정 (착공~준공~입주~대출만기) | 사용자 입력 |
| 4. 분양 계획 | 표: 분양가(확장비 구분/포함), 표: 분양대금 납입일정 | 사용자 입력 |
| 5. 자금조달 계획 | 표: 자금조달 계획 (PF/자기자본/이자율) | 사용자 입력 |
| 6. 토지 현황 | 표: 매입토지 현황 (사유지/공유지/기타, 면적/매매액/평당가) | 사용자 입력 |

### II. 사업주체 분석
| 소항목 | 주요 표/그림 | 데이터 소스 |
|--------|-----------|----------|
| 1. 회사 개요 | 표: 회사 개요 (설립일/업태/자본금/대표자), 표: 주주 현황 | 사용자 입력 + DART API |
| 2. 사업 분석 | 표: 진행중인 주택건설공사 현장, 표: 수주잔고 추이 | 사용자 입력 + DART |
| 3. 수익성 분석 | 표: 수익성 지표 추이 (8년, 매출/매출원가/영업이익/EBITDA/당기순이익) | DART 재무제표 API |
| 4. 재무안정성 분석 | 표: 재무안정성 지표 (자산/부채/차입금의존도/부채비율), 표: 유동성 지표, 표: 차입금 현황 | DART |
| 5. 현금흐름 분석 | 표: 현금흐름 추이 (영업/투자/재무활동 6년) | DART |

### III. 시장 환경 분석
| 소항목 | 주요 표/그림 | 데이터 소스 |
|--------|-----------|----------|
| 1. 부동산 주요 규제 | 규제지역/LTV/DTI/전매제한/청약 조건 | 국토부 + 정적DB |
| 2. 해당지역 인구 및 사업체 현황 | 표: 인구수·세대수 추이, 표: 연령대별 인구, 표: 산업별 사업체·종사자수, 그림: 행정구역 지도 | 행안부 주민등록 API + 통계청 |
| 3. 해당지역 부동산 시장 환경 | 표: 주택보급률·주택구성 추이, 그림: 아파트보급률, 표: 주택거래량 추이, 표: 유형별 주택분포, 표: 건축연령별 분포, 표: 면적별 공급현황, 표: 입주예정 목록, 표: 분양예정 목록, 표: 미분양 추이, 표: 미분양 단지 현황 | 국토부 MOLIT API + 통계청 |

## 3. 데이터 소스 매핑

| 카테고리 | 추출 항목 (신규 ★) |
|--------|---------------|
| 사업 개요 | 사업명, 위치, 지역지구, 시공사, 시행사, 부지면적, 건축면적, 연면적, 건폐율, 용적률, 층수, 동수, 세대수, 공사기간 |
| 분양 계획 | ★타입별(84A/84B/115OA 등) 세대수/전용면적/분양면적/평당가/세대당가/수입금, ★분양대금 납입일정(계약금/중도금/잔금), ★발코니확장비 |

### 3.3 계산 엔진 (신규)
| 엔진 | 산출물 |
|------|-------|

| ★월별 자금수지 생성기 | 48개월 월별 수입/지출/조달/상환/현금잔액 |
| ★시나리오 엔진 | 분양률 변동(기본/낙관/보수/비관 4단계) → 수지 재계산 |
| ★BEP 계산기 | 사업수지BEP/사업비Exit BEP/PF Exit BEP 분양률 |

| ★민감도 분석 | 분양률 변동 → 이익률/DSCR/미분양재고 |
| ★시세 예측 엔진 | 주변 시세 분석 + 재건축/신축 후 변동 시세 예측 |

### 3.3.1 시세 예측 엔진 (신규 — v1.1 추가)

주거용 건축(재건축/신축 모두)에 대해 **주변 시세 분석 + 준공 후 변동시세 예측** 포함.

**입력 데이터**:
- 사업지 주소 (반경 2km 내 비교 대상)
- 사업 유형 (재건축/신축)
- 준공 예정일
- 계획 분양가

**산출물**:

| 분석 항목 | 내용 | 차트 |
|--------|-----|-----|
| **현재 주변 시세** | 반경 2km 내 아파트 13개+ 단지 매매시세 (공급평당/전용평당) | 표: 매매사례 상세, 그림: 위치도 |
| **분양가 대비 프리미엄** | 분양가 → 현재 매매가 상승률 (인근 분양사례 5건+) | 표: 프리미엄(P) 현황 |
| **연도별 시세 추이** | 해당 지역 7년 평균 매매시세/분양가 추이 | 그림: 시세추이 선차트 |
| **준공 후 예상 시세** | 인근 유사 단지의 분양가→입주후 시세 변동률 기반 예측 | 그림: 시세 예측 차트 |
| **재건축 프리미엄** | (재건축 한정) 기존 아파트 시세 → 재건축 후 예상 시세, 조합원 분담금 대비 수익 | 표: 재건축 수익 분석 |

**예측 로직**:
```
1. 인근 분양사례 프리미엄 수집 (분양가 대비 현재 매매가 상승률)
   → 평균 프리미엄율 산출 (예: +2.4%)

2. 지역 매매가격지수 추이 (부동산원 REPS)
   → 연평균 상승률 산출 (최근 3년)

3. 준공시점 예상 시세 = 계획분양가 × (1 + 예상 프리미엄율)
   → 보수적/중립/낙관 3가지 시나리오

4. (재건축) 기존 시세 대비 상승 예측
   → 재건축 전 시세 / 재건축 후 예상 시세 / 조합원 분담금 / 예상 수익
```

**적용 섹션**: IV장 분양가 적정성 검토 > 5. 분양가 적정성 검토 의견 하위

## 8. 복합시설 대응 (사업 유형별)

### 8.1 지원 사업 유형

| 사업 유형 | 구성 시설 | 분양가 테이블 | 사업수지 구분 |
|---------|---------|----------|----------|
| **아파트 단지** | 아파트(다수 타입) | 타입별 | 단일 |
| **주상복합** | 아파트 + 오피스텔 + 상가 | 시설별 × 타입별 | 시설별 분리 |
| **오피스텔** | 오피스텔(다수 타입) | 타입별 | 단일 |
| **지식산업센터** | 사무실 + 상가 + 기숙사 | 용도별 | 용도별 분리 |
| **재건축/재개발** | 아파트 + 상가 (일반/조합) | 일반분양/조합분양 | 분리 |
| **생활형숙박시설** | 숙박시설 + 상가 | 용도별 | 용도별 분리 |

### 8.2 복합시설 핵심 로직

**분양수입 구분 계산**:
```
총 분양수입 = Σ(시설유형별)
  아파트: 타입별 세대수 × 세대당 분양가
  오피스텔: 타입별 실수 × 실당 분양가
  상가: 총 전용면적 × 평당 분양가
  발코니확장비: 아파트 세대수 × 확장비 단가
  중도금이자후불: 별도 산출
  VAT: 오피스텔/상가에만 적용
```

**분양률 시나리오 4단계** (분양률 기반 자동 분류):
```
기본(Base):  아파트 100% / 오피스텔 100% / 상가 100%  (분양률 100%)
낙관(Bull):  아파트  95% / 오피스텔  80% / 상가  70%  (분양률 90%+)
보수(Bear):  아파트  90% / 오피스텔  50% / 상가  50%  (분양률 70~90%)
비관(Worst): 아파트  80% / 오피스텔  30% / 상가  30%  (분양률 70% 미만)
```
→ SCR 보고서와 동일하게 **시설유형별 분양률 분리 적용**
→ 사용자 미지정 시 가중평균 분양률 기준 자동 분류

**BEP 분양률도 3종류 × 3가지 조합**:
- BEP(1): 아파트/오피스텔/상가 모두 동일 분양률
- BEP(2): 오피스텔·상가 50% 고정, 아파트만 변동
- BEP(3): 오피스텔·상가 0% 고정, 아파트만 변동

### 8.3 사업 유형 자동 감지

문서 파싱 시 키워드로 사업 유형을 자동 분류:
```
"주상복합" / "아파트+오피스텔+상가" → 주상복합
"재건축" / "정비사업" / "조합원"     → 재건축/재개발
"지식산업센터" / "산업시설"          → 지식산업센터
"생활숙박" / "레지던스"             → 생활형숙박시설
기타                              → 아파트 단지 (기본)
```

### 9.3 SCR 보고서 서술 패턴 보완

SCR 보고서는 모든 섹션에서 다음 서술 패턴을 일관되게 사용:
1. **요약 문단** (1~2줄 핵심 결론)

3. **표/그림** (정량 데이터)
4. **자료 출처 명시** (표/그림 하단에 반드시 "자료: OOO" 표기)

### 9.4 전문부(Front Matter) 상세 구조 보완

SCR 원본 전문부:
- p1: 표지 (보고서 번호, 사업명, 시행사, 로고, 세계지도)

- p3: 평가담당자 + 대표이사 직인
- p4: 전체 목차
- p5~6: 표 목차 (64개)
- p7: 그림 목차 (23개) + 부록 목차 (6개)

→ VESTRA 버전에서는 "VESTRA AI 사업성평가보고서" 브랜딩으로 대체
→ 면책조항/전제사항/유의사항은 SCR 원문을 VESTRA 맥락으로 수정하여 적용

### 9.5 V장 재무분석 전제사항 보완

SCR 원본에서 명시하는 전제사항 (기획서에 미포함):
- 분석의 전제사항 3가지 (차주 제시 사업수지 기준, 관계기관 협의에 따라 변경 가능, 본건 사업만 분석)
- 기본 가정: 분양시설/가격은 회사 분양계획 반영, 기간별 분양률 가정 필요
- 자금조달 및 신용보강: PF 이자율 가정, 공사비 20% 유보 가정
- 기타: 중도금 이자 후불제 연 4.5% 가정

→ 사용자가 이 전제사항을 미제출 시 자동 가정값 적용 로직 필요

### 9.6 BEP 분양률 3종류 정밀 정의 보완

SCR 원본의 BEP 정의:
- **사업수지 BEP**: 사업수지 = 0인 분양률 (92.4~99.0%)
- **사업비 Exit BEP**: 사업비 지출 가능한 최저 분양률. 자기자본 회수 불가 (91.2~97.7%)
- **PF Exit BEP**: PF 원리금상환 가능한 최저 분양률. 자기자본+공사비유보=신용보강 (81.4~87.2%)

→ 각 BEP에서 "자기자금 미회수", "공사비 미지급", "PF 대출 미상환" 트리거 조건 명시 필요
