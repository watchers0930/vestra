1. 분석 개요

1.1 분석 목적 및 범위

본 보고서는 VESTRA 플랫폼이 국내 프롭테크 시장에서 차지하는 위치를 객관적으로 평가하고, 경쟁 서비스 대비 강점과 약점을 식별하여 향후 전략 수립에 활용하기 위해 작성되었습니다.

분석 기준
기능 범위 — 전세 안전성 분석, 권리분석, 시세 조회, AI 분석, 사업성 분석, 보안/무결성
기술 수준 — AI/ML 알고리즘 깊이, 데이터 학습량, 자체 기술 보유 여부
시장 접근성 — 사용자 규모, 타겟 시장, 브랜드 인지도
데이터 기반 — 데이터 소스 다양성, 실시간성, 독점 데이터 보유 여부

5.2 강점·약점 종합

VESTRA 핵심 강점
1. 알고리즘 커버리지 — 26종 자체 알고리즘은 국내 프롭테크 중 가장 넓은 기술 범위
2. 유일한 사업성 분석 — BEP, DSCR, NPV, IRR, 민감도 분석을 갖춘 플랫폼은 VESTRA뿐
3. 데이터 무결성 — 블록체인 기법(머클트리 + 해시체인)으로 분석 결과 위변조 방지
4. 올인원 구조 — 전세 분석·권리·계약서·시세·세금·사업성을 하나의 플랫폼에서 제공
VESTRA 핵심 약점
1. 학습 데이터 부족 — 규칙 기반 시스템으로, 직방의 수만 건 실제 계약서 학습 대비 열위
2. 사용자 기반 부재 — 신규 플랫폼으로 인지도와 MAU에서 기존 플랫폼 대비 현격한 차이
3. 생활 정보 미제공 — 치안, 학군, 교통 등 생활 밀착형 데이터 부재
4. 대화형 AI 부재 — AI 중개사와 같은 인터랙티브 자연어 인터페이스 미구현

일반 소비자 시장에서 직방·네이버와 정면 경쟁하기보다, 이들 플랫폼이 커버하지 못하는 전문 분석 영역에 집중하는 것이 현실적입니다.
