# VESTRA 사용 가이드

---

## 슬라이드 1: 표지

**VESTRA** — AI 부동산 자산관리 플랫폼 사용 가이드
- Verified Estate Safety through Trusted Real-time AI
- "당신의 부동산을 AI가 검증하고 지킵니다"

---

## 슬라이드 2: 전체 메뉴 구성

VESTRA는 **사용자 메뉴**와 **어드민 메뉴** 2개 영역으로 구성됩니다.

| 영역 | 메뉴 수 | 대상 |
|------|---------|------|
| 사용자 메뉴 | 18개 페이지 + 전세 서브 9개 | 일반 사용자, 공인중개사, 사업자 |
| 어드민 메뉴 | 13개 탭 | 플랫폼 관리자 |

---

## 슬라이드 3: 사용자 메뉴 — 4개 그룹

사이드바 좌측 고정 메뉴, 4개 그룹으로 분류됩니다.

| 그룹 | 메뉴 수 | 목적 |
|------|---------|------|
| 메인 | 1개 | 자산 현황 한눈에 보기 |
| 분석 서비스 | 6개 | AI 기반 부동산 분석 |
| 도구 | 4개 | 조회/계산/감시 |
| 전문가 | 2개 | AI 챗봇 + 전문가 상담 |

---

## 슬라이드 4: 메인 — 대시보드

**대시보드** `/dashboard`

- 보유 자산 포트폴리오를 한 화면에서 실시간 확인
- 관리 자산 건수, 총 평가액, 평균 안전지수, 평균 리스크
- 포트폴리오 분포 차트, 최근 분석 이력
- 권리분석/계약검토/시세전망 등에서 수행한 모든 분석이 자동 수집

---

## 슬라이드 5: 분석 서비스 — 권리분석

**권리분석** `/rights`

- 등기부등본(PDF/이미지)을 업로드하면 AI가 갑구/을구 자동 파싱
- 소유권, 근저당, 가압류, 가등기 등 권리관계 시각화
- 위험도 점수(0~100), AI 종합 의견 제공
- 8대 안전진단, 권원보험 평가, 매매계약 특약 자동 생성
- SHA-256 무결성 검증으로 분석 결과 신뢰성 보장

---

## 슬라이드 6: 분석 서비스 — 계약검토

**계약검토** `/contract`

- 부동산 계약서(PDF/텍스트)를 AI가 조항별로 검토
- 계약 안전점수(0~100) 산출
- 조항별 위험도 표시 (위험/주의/안전)
- 누락 조항 자동 감지 + 맞춤 특약사항 추천
- 관련 판례 및 법규 참조 제공

---

## 슬라이드 7: 분석 서비스 — 시세전망

**시세전망** `/prediction`

- 주소 입력만으로 실거래 데이터 기반 시세 분석
- 1년/3년/5년 시나리오별 가격 전망 (낙관/기본/비관)
- 거래 분포 지도, 이상 탐지, 백테스트 결과
- 지역 비교, 회귀분석 그래프
- 신뢰도 점수와 AI 종합 의견

---

## 슬라이드 8: 분석 서비스 — 전세보호

**전세보호** `/jeonse` (하위 메뉴 9개)

- 전세 안전 진단부터 전입신고/확정일자까지 전체 절차를 안내하는 허브
- AI 전세 안전 분석: 보증금, 시세, 근저당 정보로 사기 위험도 자동 분석
- 행정 절차 가이드 5종: 전입신고 → 확정일자 → 전세권설정 → 임차권등기 → 임대차신고
- 부가 도구: 계약 체크리스트, 전세 vs 월세 비교, 이사비용 계산, 임대수익률 계산

---

## 슬라이드 9: 전세보호 — 하위 메뉴 상세

| 메뉴 | 기능 |
|------|------|
| 전세 안전 분석 | 보증금/시세/근저당 입력 → 사기 위험도 + 보증보험 가입 판단 |
| 전입신고 가이드 | 절차 4단계 + 필요서류 + 정부24 링크 |
| 확정일자 가이드 | 우선변제권 확보 절차 + 대법원 인터넷등기소 링크 |
| 전세권설정등기 | 물권화 절차 6단계 + 비용 안내(~60만원) |
| 임차권등기명령 | 보증금 미반환 시 이사 후에도 권리 유지 절차 |
| 주택임대차 신고 | 30일 이내 의무신고 → 확정일자 자동 부여 |
| 계약 체크리스트 | 거래유형별 45~50개 체크항목 + 진행률 추적 |
| 전세 vs 월세 비교 | 총 비용 비교 시뮬레이터 |
| 이사비용 계산기 | 이사비 + 중개수수료 + 세금 총합 산출 |

---

## 슬라이드 10: 분석 서비스 — 사업성분석

**사업성분석 보고서** `/feasibility`

- 사업계획서/재무제표 등 다중 문서 업로드
- 3단계 프로세스: 업로드 → 검증 → 보고서
- V-Score(0~100)로 사업성 정량 평가
- 8개 섹션 분석 (시장성/기술성/재무성 등)
- Go/No-Go 최종 판정

---

## 슬라이드 11: 분석 서비스 — 시세지도

**시세지도** `/price-map`

- 카카오 지도 위에 아파트별 시세를 색상 마커로 시각화
- 시도/구/거래유형(매매/전세/월세) 필터
- 아파트별 실거래가 목록 + 가격 변화 Top 10
- 클릭 시 공시가격/위험도 분석 연동

---

## 슬라이드 12: 도구 — 공시가격 / 세금계산

**공시가격 조회** `/official-price`
- 주소 검색 → 개별공시지가/공동주택가격/개별주택가격 3종 통합 조회
- VWorld API 실시간 연동

**세금계산** `/tax`
- 취득세/보유세/양도세 시뮬레이션
- 주택 수, 조정대상지역, 거주기간별 시나리오 비교
- 세금 최적화 인사이트 제공

---

## 슬라이드 13: 도구 — 등기감시 / API 허브

**등기감시** `/monitoring`
- 관심 부동산의 등기부 변동을 지속 감시
- 소유권/근저당 변동 시 즉시 알림
- 무결성 검증 증명서 발급 (Ed25519 서명)
- KPI: 감시 건수, 미열람 알림, 위험 등급

**API 데이터 허브** `/api-hub`
- 국토교통부/법원/VWorld 등 공공 API 연동 현황 확인
- 데이터 소스별 조회 가능

---

## 슬라이드 14: 전문가 — AI 어시스턴트 / 전문가 연결

**AI 어시스턴트** `/assistant`
- 부동산 관련 질문에 AI 실시간 답변
- 최근 분석 이력을 컨텍스트로 활용
- 관련 분석 페이지 바로가기 자동 제안

**전문가 연결** `/expert-connect`
- 공인중개사/법무사/감정평가사 등 전문가 상담 예약
- AI 분석 결과를 전문가가 직접 검증
- 상담 프로세스 인포그래픽 제공

---

## 슬라이드 15: 추가 도구

**대출 가심사** `/loan-check`
- 보증금/시세/소득 → 7대 은행 전세대출 조건 한번에 비교

**의사결정 리포트** `/decision-report`
- 대출+시세+세금+보증보험+임대인 프로파일 → Go/NoGo 종합 판정

**임대인 프로파일** `/landlord-profile`
- 임대인명으로 소유부동산, 근저당 비율, 소송 이력 종합 조회 → 안전등급(A~F)

**상호검증** `/verification`
- 임대인/임차인이 서로의 분석 결과를 검증 → 거래 투명성 강화

**AI 신뢰도** `/ai-trust`
- VESTRA AI 모델의 정확도/월별 추이/전문가 검증 결과를 투명하게 공개

## 슬라이드 16: 어드민 메뉴 — 전체 구조

어드민 페이지는 **13개 탭**으로 구성되며, ADMIN 역할 사용자만 접근 가능합니다.

| 카테고리 | 탭 |
|---------|-----|
| 현황 | 개요 |
| 사용자 관리 | 회원 관리, 인증 관리 |
| 콘텐츠 관리 | 분석 이력, 공지사항, 뉴스/정책 |
| AI/ML 관리 | ML 학습관리, 가중치 튜닝, 무결성 감사 |
| 시스템 설정 | API KEY, 보증보험 규칙, 대출 금리, 계정 설정 |

---

## 슬라이드 17: 어드민 — 현황 & 사용자 관리

**개요**
- 총 사용자 수, 역할별 분포, 인증 대기 건수
- 오늘/전체 분석 건수, 총 등록 자산, 일일 사용량 추이 차트

**회원 관리**
- 사용자 목록 조회 (검색/필터)
- 역할 변경: PERSONAL → BUSINESS → REALESTATE → ADMIN
- 일일 사용량 제한 편집, 사용자 삭제

**인증 관리**
- 사업자등록증 기반 기업/부동산 인증 심사
- 승인 시 역할 업그레이드 + 일일 한도 증가

---

## 슬라이드 18: 어드민 — 콘텐츠 관리

**분석 이력**
- 전체 사용자의 분석 요청 기록 조회
- 분석 유형별 필터 (권리분석/계약검토/시세전망 등)

**공지사항**
- 서비스 공지사항 작성/편집/삭제
- 대시보드에 자동 노출

**뉴스/정책**
- 부동산 뉴스/정책 자동 수집 현황 확인
- 알림 뉴스 확인/해제, 뉴스 삭제
- AI 채팅 컨텍스트에 자동 반영

---

## 슬라이드 19: 어드민 — AI/ML 관리

**ML 학습관리**
- 등기부등본 파싱 학습 데이터 등록/검수
- PDF 업로드 → 자동 파싱 → 사람 검증 → 승인/반려
- PII 자동 암호화(AES-256), JSONL 내보내기

**가중치 튜닝**
- 사용자 피드백 기반 분석 모델 가중치 자동 조정
- Brier Score, Log Loss, ECE 지표 추적
- 개선률/신뢰도 표시, 이전 가중치 복원 가능

**무결성 감사**
- 분석 결과의 SHA-256 해시 체인 검증
- 변조 감지 시 자동 플래그, 재검증 기능

---

## 슬라이드 20: 어드민 — 시스템 설정

**API KEY 관리**
- OAuth(Google/Naver), PG(결제), Scholar(논문검색) 등 외부 API 키 암호화 저장
- 마스킹 표시, 등록/수정/삭제

**보증보험 규칙**
- HUG/SGI 보증보험 가입 조건 규칙 설정
- 버전 관리 + 변경 이력 추적

**대출 금리**
- FSS(금융감독원) 연동 전세대출 금리 관리
- 7대 은행 금리 실시간 업데이트

**계정 설정**
- 관리자 비밀번호 변경 (8자 이상, 대소문자+숫자+특수문자 3종 조합)

## 슬라이드 21: 사용자 ↔ 어드민 연동 흐름

사용자 행동이 어드민에 실시간 반영되고, 어드민 설정이 사용자 경험에 영향을 줍니다.

**사용자 → 어드민:**
- 회원가입 → 회원 목록 반영
- 분석 실행 → 분석 이력 + 통계 업데이트
- 사업자 인증 신청 → 인증 대기 목록
- 피드백 제출 → 가중치 튜닝 데이터

**어드민 → 사용자:**
- 인증 승인 → 역할 업그레이드 + 한도 증가
- 공지사항 등록 → 대시보드 노출
- 가중치 튜닝 → 분석 엔진 정확도 개선
- 보증규칙 변경 → 보증보험 판단 로직 반영
- 뉴스 수집 → AI 채팅 컨텍스트 반영

## 슬라이드 23: 요약

**VESTRA = AI 부동산 자산관리 올인원 플랫폼**

- **5가지 AI 분석**: 권리/계약/시세/전세안전/사업성
- **30여 개 도구**: 세금/대출/수익률/비용 계산, 시세지도, 등기감시
- **전세 보호 가이드**: 5가지 행정 절차 + 체크리스트
- **어드민 13개 탭**: 사용자/분석/ML/보험규칙/금리 통합 관리
- **보안**: PII 암호화, 무결성 체인, CSRF 방어

> 부동산 거래의 시작부터 관리까지, VESTRA가 AI로 검증하고 지킵니다.
