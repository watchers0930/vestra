## 1. 문제 인식 (Problem) - 창업 아이템의 필요성

### 1-1 창업아이템 배경 및 필요성

현재 부동산 시장은 정보의 과잉과 분산으로 인해 일반인이 종합적인 판단을 내리기 매우 어려운 환경이다. 특히 전세사기 피해 규모가 최근 3년간 4조 원을 넘어서는 등 사회적 비용이 급격히 증가하고 있으나, 기존 플랫폼은 단순 매물 정보 제공에 치중하여 사전 리스크 예방 기능이 부재한다.

본 아이템은 이러한 정보 비대칭 문제 및 부동산관련 사기의 피해를 해결하기 위해 **AI 기반 독자 분석 엔진과 LLM 기술을 결합**하여 권리분석, 세무, 법무 리스크 예측을 통합 분석하는 지능형 플랫폼을 지향한다.

| 구분 | 현황 | 문제점 |
|------|------|--------|
| 부동산 정보 환경 | 정보 과잉/분산 | 종합적 판단 어려움 |
| 법/세무 구조 | 전문 지식 의존 | 일반인의 접근성 낮음 |
| 기존 플랫폼 | 매물 중심 서비스 | 의사결정 지원 한계 |
| 전문가 의존 | 비용/시간 부담 | 대중적 활용 어려움 |
| 리스크 관리 | 사후 대응 위주 | 사전 예방 기능 부재 |

본 사업은 부동산 거래 과정에서 발생하는 정보 비대칭과 복잡한 의사결정 문제를 AI 기술로 해결하여, 개인과 기업 모두가 신뢰할 수 있는 합리적 판단 환경을 제공하는 것을 목표로 한다.

| 구분 | 현황 |
|------|------|
| 사회적 필요성 | 전세사기, 부동산 분쟁 예방을 통한 사회적 비용 절감 |
| 시장적 필요성 | 부동산 가치예측을 통한 실수요자 중심의 합리적 의사결정 지원 |
| 기술적 필요성 | AI 기반 통합 분석 기술을 활용한 자동화 수요 증가 |
| 정책적 필요성 | 디지털 전환 및 데이터 기반 행정 방향과 부합 |

이에 본 사업은 AI 기술을 활용하여 부동산 권리/계약/세무/법무 리스크를 통합 분석하는 지능형 플랫폼을 구축함으로써, 기존 시장의 구조적 한계를 해결하고자 한다.

### 1-2 창업아이템 목표시장(고객) 현황 분석

**1. 국내 부동산 시장 규모 및 구조**

국내 부동산 시장은 여전히 국내 최대 규모의 자산 시장으로, 거래 규모와 연관 산업 파급력이 매우 크다.
- 국내 부동산 자산 총액: **약 6,000조 원** 이상
- 연간 부동산 거래 금액: **약 1,800조 원** 수준
- 주택 거래 중 개인 간 거래 비중: **80% 이상**(부동산 중개)

이처럼 거래 규모는 거대하나, 실제 거래 판단은 여전히 개인의 경험과 제한된 정보에 의존하고 있어 **정보 비대칭성이 매우 큰 시장**이다.

**2. 부동산 거래 리스크 증가 추세**

최근 5년간 부동산 시장에서는 다음과 같은 위험 요인이 지속적으로 확대되고 있다.

| 구분 | 현황 |
|------|------|
| 전세사기 피해 | 최근 3년간 피해 규모 약 **4조 원 이상** |
| 분쟁 조정 건수 | 연평균 **15% 이상 증가** |
| 세무 리스크 | 다주택/양도세 관련 과세 민원 급증 |
| 법률 분쟁 | 부동산 관련 민사 소송 비중 지속 증가 |

**3. 디지털/AI 전환에 따른 시장 변화**
- 국내 AI 시장 규모: 연평균 20% 이상 성장
- 정부 디지털 전환(DX) 정책에 따라 공공/민간 AI 도입 가속
- 부동산 분야에서도 데이터 기반 의사결정 요구 급증

본 사업은 시장 내 분절된 자산/부동산/법률 정보 구조로 인해 발생하는 의사결정 비효율 문제를 해결하고, AI 기술 성숙을 기반으로 통합적/실행 가능한 자산 의사결정 플랫폼을 구현함으로써 새로운 시장 가치를 창출하는 것을 목표로 한다.

### 1-3 SWOT분석

**Strength**
- **독자 엔진 기술 보유**: 순수 TypeScript 기반 3개 핵심 엔진(등기부등본 파싱 592줄, 4단계 검증 700줄, 리스크 스코어링 511줄) - AI에 의존하지 않는 독자 기술
- 전문가 지식과 AI의 결합 모델: 법/세무 전문성 + 기술력
- 차별화된 기능 세트: 단일 플랫폼에서 7가지 분석 제공(권리분석, 계약검토, 세무시뮬레이션, 가치예측, 전세보호, AI 어시스턴트, API 데이터 허브)
- **MVP 이미 개발 완료 (v0.7.8)**: Vercel 배포 및 운영 중
- 초기 시장 기회: 경쟁사 미진출 영역 다수

**Weakness**
- 초기 브랜드 인지도 부족: 신생 스타트업의 한계
- 대규모 데이터 확보 전까지 분석 정밀도 제한: 초기 정확도 96~97%
- 한정된 팀 역량: 초기 조직 규모 소규모

**Opportunity**
- 전세사기 증가로 인한 사전 검증 수요 확대: 시장 요구 증대
- 정부의 디지털 전환 정책 및 AI 육성 기조: 지원 정책 우호적
- 부동산/금융 융합 서비스 수요 증가: Wealthtech 시장 성장
- 정부 초기창업패키지 등 지원 사업 확대

**Threat**
- 대형 플랫폼의 유사 서비스 진입 가능성: 시장 선점 필요
- 법/제도 변화에 따른 서비스 제약 가능성: 규제 리스크
- 거시경제 침체로 부동산 거래 위축 가능성
- 데이터 개인정보보호법 강화: 규정 준수 필요

## 2. 창업 아이템의 개발 계획

### 2-1 개발 전략 개요

본 사업은 이미 **MVP 개발을 완료(v0.7.8)**하고 상용화 직전 단계에 진입하였다.
- 3개 독자 TypeScript 엔진(등기부등본 파싱, 4단계 검증, 리스크 스코어링) 개발 완료
- 부동산 실거래/등기/법령/판례 등 공공 데이터 활용 가능
- 상용 LLM API(OpenAI GPT-4.1-mini) 활용, 비용 최적화 운영 중
- Vercel 클라우드 배포, Neon PostgreSQL 데이터베이스 운영 중

## 4. 성장전략 (Scale-up) - 사업화 추진 전략

### 4-1 단계적 기능/자산 확장을 통한 사업화 추진전략

본 사업은 초기부터 모든 기능과 자산을 포괄하는 방식이 아닌, **시장 수용성과 기술 안정성을 고려한 단계적 기능/자산 확장 전략**을 기반으로 사업화를 추진한다.

**1단계(Year 1-2): 부동산 자산 중심**
- 초기에는 부동산자산을 중심으로 실거래/규제/공간/인구 데이터를 통합 분석하는 핵심기능 구현 (✅ MVP 완료)
- 명확한 문제해결 효과입증
- 초기 고객확보 및 시장반응 검증

**2단계(Year 2-3): 자산범위확장**
- 금융자산, 법인자산, 계약/권리자산으로 분석범위 확장
- 서비스의 활용도와 고객 당 가치(ARPU) 점진적 확대
- 기술개발리스크최소화 + 기존데이터/AI 모델 재활용으로 비용 효율화

**3단계(Year 3+): 플랫폼확장**
- 자산범위확장은 고난도의 의사결정수요를 자연스럽게 유입
- 고액자산가, 패밀리오피스, 법인고객층 진입
- 중장기 성장 동력 확보

### 4-2 고객군 고도화를 통한 사업화 추진전략

본 사업은 단순 사용자 수 확대가 아닌, 고부가가치 의사결정이 요구되는 고객군으로의 단계적 고도화를 핵심 사업화 전략으로 설정한다.

**1단계: 개인 투자자 및 전문 투자자**
- 초기에는 개인 투자자 및 전문 투자자를 대상으로 서비스 제공
- 시장반응과 활용성 검증
- 초기 매출 창출 및 사용자 데이터 축적

**2단계: 기관고객(자산운용사, 법무/회계법인)**
- 자산구조분석, 법률리스크관리, 의사결정 시나리오 제공
- 차별화된 기능을 중심으로 서비스가치 재정의
- 계약단가상승 + 수익안정성확보

**3단계: 고액자산가, 패밀리오피스, 법인고객**
- 단기사용서비스 → 장기적 파트너십 기반의 사업관계 형성
- 고객지속성극대화

## 5. 마케팅방안

본 사업의 마케팅 전략은 단순 노출이나 홍보 중심이 아닌, 자산/부동산 의사결정이 실제로 발생하는 시점에 신뢰 기반 솔루션을 제시하는 정밀 타깃 전략을 기본 방향으로 한다.

### 1. B2B 마케팅 전략

| 구분 | 내용 |
|------|------|
| 핵심 타깃 | 자산관리사, 부동산 자산운용사(AMC), 법무/회계 법인, 부동산/금융 컨설팅사 |
| 고객 특성 | 반복적인 자산/부동산 의사결정 수행, 리스크 회피 및 업무 효율성 중시 |
| 마케팅 메시지 | "AI 기반 사전 리스크 분석을 통한 의사결정 효율 및 법적 안정성 확보" |
| 접근 전략 | 문제 기반 세일즈(Problem-driven Sales), 신뢰 중심 컨설팅형 접근 |
| 주요 채널 | 전문가 네트워크, 산업 세미나/웨비나, 직접 영업, 파트너 추천 |
| 전환 방식 | 데모/시범 적용 → 구독 계약 체결 |
| 핵심 KPI | 유료 기관 수, 계약 전환율, 기관당 평균 매출(ARPA) |

### 2. B2C 마케팅 전략

| 구분 | 내용 |
|------|------|
| 핵심 타깃 | 전/월세 계약 예정자, 개인 투자자, 임대인 및 소규모 자산 보유자 |
| 마케팅 메시지 | "계약 전 5분, AI로 리스크를 먼저 확인하세요" |
| 접근 전략 | 위험 인지 시점 집중 공략, 콘텐츠 기반 신뢰 형성 |
| 주요 채널 | 부동산 정보 플랫폼, 온라인 커뮤니티, 콘텐츠 마케팅, 검색 유입 |
| 전환 방식 | 무료 체험 → 건별 분석 과금 또는 구독 |
| 핵심 KPI | 월간 활성 사용자(MAU), 무료→유료 전환율, 재이용률 |

### 3. 전문가 실증단 기반 검증 마케팅 전략

본 사업은 체험단/후기 중심의 일반적인 홍보 방식이 아닌, 전문가 실증단을 활용한 검증 중심 마케팅 전략을 추진한다. 서비스 초기 단계에서 부동산/법률 실무자 및 관련 전문가 10인 내외로 실증단을 구성하여, 실제 계약/투자 사례를 기반으로 AI 분석 정확도와 활용성을 검증한다.

### 4. 단계별 마케팅 실행계획

사업 초기 6개월 동안에는 전문가 실증단 10곳과 개인 이용자 100명 이상을 확보하여 최소 30건 이상의 실제 분석 사례를 축적하는 것을 목표로 한다.
