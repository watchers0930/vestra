1. 서비스 개요

1.1 플랫폼 소개

VESTRA는 부동산 거래의 핵심 리스크를 AI로 자동 분석하는 B2B SaaS 플랫폼입니다. 등기부등본 파싱, 권리관계 분석, 계약서 검토, 시세 예측, 전세 안전성 평가, 세무 시뮬레이션 등 부동산 거래 전 과정을 하나의 플랫폼에서 처리합니다.

1.2 서비스 URL

https://vestra-plum.vercel.app (프로덕션 배포 완료)

3. 핵심 기술 분석

3.1 AI 기반 분석 엔진 (5개 모듈)

OpenAI GPT-4o 모델을 활용한 부동산 전문 AI 분석 시스템. 각 모듈별 도메인 특화 시스템 프롬프트를 설계하여 법률/금융 전문가 수준의 분석을 제공합니다.

모듈입력출력특징
권리분석등기부등본 텍스트위험 요소, 안전점수, 권리관계 도식갑구/을구 자동 분류
계약검토매매/임대 계약서위험 조항, 누락 항목, 종합 평가조항별 중요도 분류
가치예측주소, 면적, 실거래 데이터3가지 시나리오 예측, 변수 분석국토부 API 연동
전세보호계약 정보 (보증금, 기간 등)전세권 설정 필요도, 법률문서 생성자동 문서 생성
AI 상담자유 대화부동산 전문 답변스트리밍 응답

3.2 자체 개발 엔진 (AI 미사용, 4개 모듈)

AI 의존도를 줄이고 비용 효율성을 확보하기 위해, 규칙 기반 로직은 순수 TypeScript로 자체 개발했습니다. 이 엔진들은 단위 테스트로 검증됩니다.

모듈파일기능테스트
등기부 파싱 엔진registry-parser.ts정규식 기반 등기부등본 구조화 파싱 (표제부/갑구/을구 분리, 금액 추출, 권리자 식별)27개
리스크 스코어링risk-scoring.ts100점 감점 방식, 12개 위험요소 자동 평가 (근저당비율, 가압류, 소유권 변동 등)15개
4단계 검증 엔진validation-engine.ts포맷 검증 → 산술 검증 → 문맥 검증 → 크로스체크 (등기부 정합성 자동 검증)12개
세금 계산기tax-calculator.ts취득세/보유세/양도세 자동 계산 (생애최초, 다주택 중과, 1세대1주택 비과세 적용)15개

설계 원칙: AI와 규칙 기반 로직을 명확히 분리하여, AI 비용이 발생하지 않는 기능은 자체 엔진으로 처리합니다. 이를 통해 API 비용 90% 절감, 응답 속도 10배 향상, 오프라인 동작 가능성을 확보했습니다.

3.3 공공 데이터 연동

국토교통부 실거래가 공공 API를 연동하여 실시간 시세 데이터를 제공합니다. XML 응답 파싱, 법정동 코드 매핑, 에러 핸들링을 자체 구현했습니다.

4.2 보안 심층 방어 (Defense in Depth)

계층기술설명
L1. HTTP 헤더6종 보안 헤더X-Frame-Options: DENY, HSTS, X-XSS-Protection 등
L2. 인증Middleware세션 쿠키 기반 API 접근 제어
L3. 속도제한Rate Limiter인증 유저 30req/min, 비인증 10req/min
L4. 입력살균SanitizerHTML 태그, script, XSS 패턴 제거
L5. 비용가드Cost Guard사용자당 일일 50회 OpenAI API 호출 제한
